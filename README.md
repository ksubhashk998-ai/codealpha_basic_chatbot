# codealpha_basic_chatbot

🤖 CodeAlpha Python Programming Internship – Task 4

Basic Chatbot

📌 Description

The Basic Chatbot is a simple rule-based chatbot developed using Python. It responds to predefined user inputs such as “hello”, “how are you”, and “bye” using conditional statements.

This project is designed to help beginners understand basic conversation logic and control flow in Python.

⚙️ Functionality

Accepts user input from the console

Matches user input using if-elif-else conditions

Displays predefined responses

Terminates the chat when the user enters "bye"

🧠 Concepts Used

Python Functions

if-elif-else statements

while loop

User input and output

▶️ How to Run

Install Python 3.x on your system

Open Terminal / Command Prompt

Navigate to the project folder

Run the program using:

python chatbot.py

💬 Example Interaction

Input:

hello how are you bye

Output:

Hi! Nice to meet you. I'm fine, thanks for asking! Goodbye! Have a great day.
